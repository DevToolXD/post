🚀 북마클릿 반복 요청 생성기

Network 탭에서 복사한 fetch 코드를 붙여넣으면, 원하는 횟수만큼 반복 실행하는 북마클릿을 자동으로 만들어줍니다.

📄 생성된 북마클릿 코드

아래 코드를 Ctrl+D → URL 칸에 붙여넣고 저장하세요. 그리고 대상 사이트에서 북마크를 클릭하면 실행됩니다.

여기에 북마클릿 코드가 나타납니다...

📖 사용 방법

  1. 개발자 도구(F12) → Network 탭 → 원하는 요청 우클릭 → Copy → Copy as fetch
  2. 이 페이지에 붙여넣고 횟수, 간격 설정
  3. "북마클릿 생성" 클릭
  4. 생성된 긴 코드를 복사
  5. 대상 사이트에 접속 → Ctrl+D → URL 칸에 코드 붙여넣고 저장
  6. 저장한 북마크 클릭 → 실행!
⚠️ 주의: 반드시 대상 사이트의 페이지 내에서 북마크를 클릭해야 정상 작동합니다. (CORS 회피)
너무 빠른 간격(0ms)이나 과도한 요청은 IP 차단의 위험이 있으니, 교육/테스트 목적으로만 사용하세요.
